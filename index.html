ABOUT THIS PROJECT

This expense tracking system project will keep the records of
expenses of any user on a day to day basis. User will input
their expenses one by one, the database will store the records.
There are many features on this project, main feature is to have a a full statistics report system.
Main focus of the project is to minimize any users regular expenses.
